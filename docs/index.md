# Flan: Turbulent Trace Impurity Transport Code

Welcome to the documentation for **Flan**, a lightweight impurity and neutral
transport code designed for edge plasma physics, SOL modeling, and rapid
numerical experimentation.

This site contains:

- **User documentation** for running Flan  
- **Physics notes** describing the models implemented  
- **Developer documentation** for contributing or extending the code
