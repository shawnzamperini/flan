# Getting Started

This guide walks you through the basics of downloading, compiling, and running FLAN.

## Requirements

- C++20 compiler (GCC, Clang, or NVHPC)
- CMake ≥ 3.20
- MPI implementation (OpenMPI or MPICH)
- OpenMP support
- HDF5 (optional, depending on background data format)
